INSTALL INSTRUCTIONS:

Clone MovieReview repository to repository on your local computer.
Copy MovieReview folder into htdocs folder of xampp. 
Start up xampp.
Within shell, navigate to xampp/htdocs/MovieReview and begin mysql.
Type "source movies.sql".  
Within your internet browser, navigate to http://localhost/MovieReview/freestyle/index.php to take you to the homepage.
To access the writeReview.php page, you must login with the username 'admin' and the password '54321'.
